# YouTube Clone

A fully responsive YouTube front-end clone built as a Semester 2 web development project. This application replicates the core user interface of YouTube, featuring a video player layout, sidebar navigation, search bar, and dynamic category feeds.

## ✨ Features
* **Responsive UI:** Fully optimized for desktops, tablets, and mobile screens.
* **Video Player Component:** Clean layout for watching videos with titles and descriptions.
* **Dynamic Feed:** Content organized by categories and tags.
* **Modern Sidebar:** Toggleable navigation mimicking the real YouTube experience.
* **Search Bar:** Functional header UI for video discovery simulation.

## 🛠️ Tech Stack
* **Languages:** JavaScript (ES6+), HTML5, CSS3
* **Icons:** FontAwesome

## 🎓 Academic Context
* **Semester:** Semester 2 Project
* **Objective:** Practical application of Java Script and modern CSS styling.
